### ✨ Key Features
* **감정 기반 게이미피케이션**: 봉사 활동 시 경험치(EXP)를 획득하여 '곰돌이 아바타'를 성장시키는 시스템으로 지속적인 참여 동기 부여
* **지도 기반 봉사 탐색**: 위치 기반 서비스를 통해 내 주변의 봉사 공고를 시각적으로 확인하고 직관적으로 신청
* **통합 봉사 관리 시스템**: 카카오 로그인 연동, 모집/후기 게시판, 지원자 관리 및 출석 체크를 통한 체계적인 봉사 이력 관리
* **필코노미(Feel-conomy) 서비스**: 단순 활동을 넘어 사용자의 감정적 만족과 사회적 기여도를 시각화하여 선한 영향력 확산

---

### 👩‍💻 My Role (Frontend & Design)
* **UI/UX 디자인**: Figma를 활용하여 곰돌이 아바타 시스템을 포함한 서비스 전반의 인터페이스 및 사용자 경험 설계
* **프론트엔드 퍼블리싱**: HTML5와 CSS3를 활용하여 지도 탐색 화면, 마이페이지, 게시판 등 서비스 전체 웹 페이지 구현
* **사용자 경험 최적화**: 감정 기반 서비스 특성에 맞춰 따뜻하고 직관적인 컬러 시스템 및 레이아웃 구축

---

### 🛠 Tech Stack
* **Frontend**: HTML5, CSS3
* **Backend**: Spring, Thymeleaf
* **Infrastructure**: AWS
* **Design**: Figma
